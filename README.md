# 縮網址小工具 Short URL tools

這是一個簡單的縮網址網站，使用者可以貼上原始網址併產生對應的一組新網址。
在瀏覽器貼上該網址即可跳轉到原網址。

## 技術架構

- **前端**：`Vue 3` + `Tailwind CSS`
- **後端**：`Node.js` + `Express`
- **資料庫**：`Supabase` (`PostgreSQL`)
- **部署平台**：`Zeabur`
- **版本控制**：`Git`
